Babel Canvas

Como o o javascript avança muito rápido os navegadores não conseguem acompanhar o rítmo. Por isso o babel é uma ferramenta que possibilita um pré-processamento do código e converte ele para uma forma que até os navegadores mais antigos podem entender.

Esta aplicação é uma instalação básica do Babel. Após instalação via yarn o comando yarn dev vai transformar qualquer javascript moderno do arquivo main.js em um javascript universal dentro do main.js

Dependencias:

  • Yarn
  • Node
