Usando Classes

Classes são como templates para a criação de objetos. Elas nos permite trabalhar no javscript com modelos de herança comparável a algumas linguagens orientadas aS objeto.

O exemplo1.js cria uma classe chamada List contendo um array data e um método add() que adiciona um item a este array. Em seguida extende esta Classe para uma nova classe Tarefas, incluindo nela uma descrição.

O exemplo2.js cria uma classe usuario e extende ela em uma classe chamada Admin com um paramento adicional admin:true. Na classe usuairo há também um metodo que verifica que o usuario é ou não admin
