Usando Classes

Dependencias: Yarn, Node, Babel

Classes são como templates para a criação de objetos. Elas nos permite trabalhar no javscript com modelos de herança comparável a algumas linguagens orientadas aS objeto.

Esta aplicação cria uma classe chamada List contendo um array data e um método add() que adiciona um item a este array.

Em seguida extende esta Classe para uma nova classe Tarefas, incluindo nela uma descrição.
